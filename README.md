# rtow

My walkthrough of [_Ray Tracing in One Weekend_](https://raytracing.github.io/books/RayTracingInOneWeekend.html)

I've added unit tests.

This whole graphics thing? Kinda neat ey?

## Requirements

- GNU make and bash
- g++ or clang

## Targets

Tests:
```bash
$ make run_tests
```

Generate image/main:
```bash
$ make run
```
